

Playing Card - Gimp Filter von TinTran
Kostenloses Gimp Script um Spielkarten
mit eigenen Motiven zu erstellen.

########################################################



Ich habe hier ein kostenloses Script von TnTran zum Download für sie.
Mit diesem Tool können sie Kartenspiele mit eigenem Dekor erstellen. 
Das Aussehen der Spielkarten kann von ihnen im Scriptmenü  beeinflusst 
werden. Alle mit dem Tool erzielten Ergebnisse können über 
die erstellten Ebenen weiter bearbeitet und an ihre Wünsche angepasst 
werden. Der Filter funktioniert auch mit der Gimp Version 2.10.2 problemlos. 
Auch unter unserer Testinstallation hat das Script keinerlei Probleme 
verursacht. Wenn sie ihrer Fantasie etwas Freiraum geben, können wirklich 
tolle Filterausgaben erstellt werden. Für diesen kostenlosen Filter geht
natürlich noch ein dickes Dankeschön an den Ersteller TinTran.




Anleitung:
1. Öffnen sie ein Bild, aus dem sie eine Spielkarte machen möchten.

2. (Optional, aber es wird die Karte besser aussehen lassen). 
   Zuschneiden / Größe des Bildes mit der Dimension 688 x 1116.
   Siehe Auswählen eines Rechtecks mit einem bestimmten Seitenverhältnis. 
      
3. Run Skript (Diego / Playing Card...). Wählen Sie Ihren Kartenwert aus, wählen 
   Sie die Farbe der Karte und wählen Sie die Schriftart. OK klicken".
   
4) Eventuell weitere Einstellungen vornehmen. (Motiv spiegeln/Karte mit Schatten)

5. Das war es, jetzt wird eine Spielkarte basierend auf Ihren Einstellungen erstellt.

PlayingCards mit eigenem Dekor.




Spielkarten selber erstellen, ein neues kostenloser Gimp Filter von TinTran.
Mit diesem Tool können Spielkarten mit eigener Dekoration erstellt werden.
Unter unserer Gimp Installation und auch unter der neuen Version 2.10.2
hat das Tool keine Probleme bereitet oder Fehler produziert. Unsere Tester 
waren von den Filterausgaben sehr angetan. Alle mit diesem Tool erstellten
Dateien können über die Ebenen weiter bearbeitet und an ihre Wünsche angepasst 
werden. Im ausführlichen Filtermenü kann die Ausgabe auch schon sehr schön 
angepasst werden. Für diesen tollen kostenlosen Filter noch einmal ein 
Dankeschön an TinTran.





Sollten nach dem Download dieser Datei Probleme bei der Installation oder
der Arbeit mit diesem Tool auftreten, empfehlen wir ihnen den Blog Artikel 
noch einmal aufmerksam durchzulesen. Unsere Erfahrung zeigt, dass die meisten 
Probleme nach dem Lesen beantwortet worden sind. Ist das aber nicht der Fall,
